1. 배열 Array

Array의 정의

배열은 연관된 데이터를 모아서 관리하기 위해서 사용되는 데이터 타입이다. 변수가 하나의 데이터를 저장하기 위한 것이라면 배열은 여러 개의 데이터를 저장하기 위한 것이다.

Array의 설명

key값과 value값을 일일히 지정해줘야 객체와 다르게 key값을 지정해주지 않아도 값을 나열할 수 있도록 해주는 것이 배열이다.

Array의 예시

사람마다 여러가지 정보를 기록해야하는 생활기록부가 객체라면 단순히 순서 관계 없이 나열 가능한 급식 메뉴같은것이 배열이라고 할 수 있다.

Array의 코드

var menu = ["오곡밥", "배추김치", "제육볶음", "된장국", "나물"]; //menu라는 배열에 오곡밥, 배추김치 등을 대입

배열은 순서를 따지지 않고 각각의 값에 Key값도 존재하지 않는다.

2. 색인 Index

Index의 정의

추가적인 쓰기 작업과 저장 공간을 활용하여 데이터베이스 테이블의 검색 속도를 향상시키기 위한 자료구조이다.

Index의 설명

배열의 요소를 찾을 때 사용하는 index는 배열[인덱스넘버]의 방식으로 사용 할 수 있다. 배열에서 N번째를 찾는데 사용하는 것으로 N은 사람과 달리 1부터가 아니라 0부터 숫자를 센다.

Index의 예시

2학년 1반의 출석부라는 배열에서 25번째에 있는 사람을 찾고 싶을 때에 사용한다.

Index의 코드

var gc21 = ["김경근", "나진성", "민지환", "박병혁", "서정범"]; //gc21라는 배열에 다음과 같은 값들을 대입
console.log(gc21[2]); //gc21이라는 배열에서 3번째(인덱스넘버 이기 때문에 2라도 3번째를 가르킨다) 요소를 콘솔로 띄워줘

배열에서 특정 번째 값을 찾기 위해서 사용한다.

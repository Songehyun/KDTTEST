1. 함수 Function

Function의 정의

자신의 외부 코드가 호출할 수 있는 하위 프로그램이다. 명령문의 시퀀스로 구성된 함수 본문을 가지며 함수에 값을 전달하면 함수는 값을 반환할 것이다.

Function의 설명

만들고 싶은 다양한 기능을 가능하게 해주는 것이 함수이다. 행동에 따라서 무엇인가 값이 반환되는것이 함수이다.

Function의 예시

회사의 사장이 직원 A에게 아침 10시마다 커피를 가져다 달라고 하는것을 함수라고 볼 수 있다. 다음의 조건을 출근하고 1시간 뒤로 바꿔도 출근 시간이 오전9시까지라면 똑같이 동작할 것이다.

Function의 코드

var time = prompt("지금시간은?", 0) // 변수 time에 시간을 입력받기 위해서 안내문을 띄우는 것, 기본값은 0이다.
if(time == 10) {document.write("커피먹을시간")} // time이 10일경우 커피먹을시간이라는 단어를 출력

구현하고 싶은 다양한 작업들을 함수로서 구현할 수 있다.

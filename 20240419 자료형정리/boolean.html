1. 불리언 Boolean

Boolean의 정의

불, 불린, 불리언 이라고 불리며 대문자로 시작하는 True나 False를 선언하고 이용하는 자료형이다.

Boolean의 설명

참과 거짓을 가려내는데 주로 사용하는 자료형으로 참 거짓을 확실히 나눌수 있는 상황에 사용하기 때문에 사람이 확실하게 정해놓을 수 있다.

Boolean의 예시

1+1은 2이다 = True , 2+2는 22이다 = Flase

Boolean의 코드

var dog = { ispeople: False, isdog: True }; //개를 생각하고 만든 dog라는 객체 안에 ispeople, isdog 라는 불리언 함수를 사용하기 위한
key값을 대입하였다. ispeopel 는 거짓, isdog는 참이다.

참, 거짓이라는 확실하게 나눠진 조건을 사용하기 때문에 사람이 내용을 확실하게 표현할 수 있다.

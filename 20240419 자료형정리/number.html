1. 정수 integer

integer의 정의

마이너스, 플러스 부호가 붙어있는 자연수를 integer, 정수라고한다. 줄여서 int로 불린다. interger의 기본값은 0이다.

integer의 설명

0.xxx 처럼 소수점 아래로 숫자가 붙지 않은 모든 숫자는 integer, 정수다.

integer의 예시

지나가는 사람 1명, 떨어진 사과 3개, 오늘의 식사 메뉴 5종류 모두 1, 3, 5 라는 정수이다.

integer의 코드

var a = 5; //변수 a에 정수 5를 대입
var b = 10; //변수 b에 정수 10을 대입
var plus = a + b; //변수 a 와 b를 더해줘

원래라면 단어인 a와 b를 더하지 못하는게 정상이지만 여기서 a와 b는 숫자, 정수로 여겨져서 값을 확인해 보면 두 숫자를 더한 값인
15가 나온다.

2. 소수 float

float의 정의

실수, float는 int와 달리 소수까지 표현 가능하며, floating point에서 유래되었다. 한글로 번역하면 떠돌이 소수점 이라는 뜻으로 소수점이 떠돌아 다니는데 컴퓨터가 12.34라는 소수점 숫자를 표현할때 숫자인 1234를 기억해 둔 뒤 2와 3사이의 점 위치를 찍는 방식이다.

float의 설명

0.xxxx 부터 평범한 숫자인 123까지 숫자로 나타낼 수 있는 모든 수를 실수라고하며, float이라고 한다.

float의 예시

3학년 수학 평균 점수는 77.5점, 영어 평균 점수는 80점, 국어 평균 점수는 82.5점일 경우 77.5 , 80, 82.5 모두 실수이다.

float의 코드

var math = 77.5; //변수 math에 실수 77.5를 대입
var english = 80; //변수 english에 실수(이며 정수인) 80을 대입
var plus = math + english; //변수 math 와 english를 더해줘

정수와 마찬가지로 원래라면 더해지지 않을 두 문자가 실수값을 넣은 것으로 인해 덧셈 계산이 가능해진다.

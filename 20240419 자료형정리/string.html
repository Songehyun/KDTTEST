String Data Type

1. 문자열 String

String의 정의

문자열, String은 기호의 순차 수열을 말한다. 이러한 기호는 미리 정의된 집합이나 음소 문자에서 선택한다.

String의 설명

문자와 다르게 a , p , p 한개씩 변수에 넣을 수 있는 게 아닌 apple 단어를 한번에 넣을 수 있도록 되어있는 것이 문자열 string이다.

String의 예시

a,p,p는 문자 apple , studnet 을 문자열이라고 한다.

String의 코드

var apple = "apple"; var pie = "pie"; // 변수 apple에 apple라는 문자열을 변수 pie에 pie라는 문자열을 대입하였다.
var applepie = apple + pie // 문자열을 담은 변수 apple과 pie를 합친다.

이렇게 할 경우 문자열은 이어져서 출력된다.

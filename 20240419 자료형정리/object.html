1. 객체 Object

Object의 정의

객체 Object는, 여러 데이터를 담을 수 있다는 점에서 배열과 비슷하나, 배열보다 담을 수 있는 데이터가 더 다양하다. 중괄호{}를 이용하여 객체를 생성할 수 있으며, 이 중괄호 내부에 Key:value 로 구성해 값을 여러개 넣을 수 있다. 이런 데이터 쌍 하나를 Property 프로퍼티 라고 부른다.

Object의 설명

객체를 선언하고 중괄호 안에 키와 값을 이용해서 데이터를 만들어 가는 방식으로 작성, 내용물들을 key값으로 구분해야 하는 경우에 사용한다.

Object의 예시

학교에서 5학년 2반의 정보를 기록한 서류가 있다면 그 서류에는 반의 학생수가 몇명인지 이번년도 담임 선생님은 누가 배정이 되었는지 등의 정보가 있을것이다. 이 서류를 객체라고 한다.

Object의 코드

var class52 = {student:20, classname:"5학년2반", teacher:"김경근"}; //class52라는 객체에 Key값으로 student, classname, teacher가
있으며, 각각의 value로서 값들이 대입되어 있다.

객체는 중괄호 안에 Key:value 구성을 다양하게 넣어두는 것이다.

2. 객체의 속성 : Static 정적

Static 정적 속성의 설명

Key + value값, 즉 Property가 객체 안에 정해져 있는 경우를 정적인 객체 라고 말한다.

Static 정적 속성의 예시

미리미리 전부 정해져있는 오픈한 식당의 메뉴판 같은것이 정적 객체라고 할 수 있다.

Static 정적 속성의 코드

var kimbobmenu = { je6:{name:"제육덮밥", price:4000}, kimbob:{name:"치즈김밥", price:2000}};
//kimbobmenu라는 객체에 je6이라는 key를 가진 값과 kimbob이라는 key를 가진 값을 대입했다.

3. 객체의 속성 : Dynamic 동적

Dynamic 동적 속성의 설명

Key + value값, 즉 Property가 객체 안에 정해져 있는 경우가 아니라 객체 밖에서 새로새로 추가되는 형태를 말한다.

Dynamic 동적 속성의 예시

새로온 전학생의 명부를 추가할 경우가 동적인 경우라고 볼 수 있다.

Dynamic 동적 속성의 코드

var class32student = {kimminjae:{name:"김민재", age:"18"}}; //class32student라는 객체에 현재 하나의 내용만이 존재한다.
class32student.newpeople = {};
class32student.newpeople.name = "새친구"; // 전학생의 정보를 동적 할당한다.

현재 당장 정하지 못하거나 나중에 추가되는 경우에 동적 할당을 이용한다.

4. 객체의 초기화 : Initialize

Initialize의 설명

어떤 일을 시작하기 전에 준비를 하게 되는데 이것을 다른 말로 초기화라고 한다. 객체 지향 프로그래밍도 초기화에 해당하는 기능이 제공되는데 이것을 생성자(constructor)라고 한다. 즉 객체를 생성할 때 항상 실행되며, 맨 처음 실행되는 메소드이다.

Initialize의 예시

새 식당을 오픈한 경우 그 식당의 상태를 초기화라고 한다. (다 삭제하고 초기로 되돌리는게 아니라 초기 세팅이라는 뜻)

Initialize의 코드

var newstore = {}; // 객체 newstore를 초기화했다.
